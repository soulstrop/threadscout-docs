Runbook: Registering OIDC Providers

This document provides the exact technical metadata required to register ThreadScout as a “Relying Party” with various identity providers. Consistency in these strings is critical to preventing Authentication Loops.

1. Global Application Metadata

Use these values for the “General Settings” or “OAuth Consent Screen” section of any developer console.

  • App Name: ThreadScout
  • Homepage: https://threadscout.constans.dev
  • Privacy Policy: https://threadscout.constans.dev/privacy
  • Terms of Service: https://threadscout.constans.dev/about
  • Support Email: (Use your verified admin email)
  • Authorized Domain: constans.dev

2. Redirect URIs (NextAuth Standard)

NextAuth.js expects the callback to follow a specific pattern. You must register the exact string below for each provider.

Provider Redirect URI (Production / Live)
Google https://threadscout.constans.dev/api/auth/callback/google
Apple https://threadscout.constans.dev/api/auth/callback/apple
GitHub https://threadscout.constans.dev/api/auth/callback/github
eBay https://threadscout.constans.dev/api/auth/callback/ebay
Facebook https://threadscout.constans.dev/api/auth/callback/facebook
PocketID (Internal) https://threadscout.constans.dev/api/auth/callback/pocketid

3. Provider-Specific Console Instructions

Google (Google Cloud Console)

  1. Navigate to APIs & Services > Credentials.
  2. Create an OAuth 2.0 Client ID.
  3. Authorized JavaScript origins: https://threadscout.constans.dev
  4. Scopes: openid, email, profile.

Apple (Apple Developer Portal)

  1. Create an App ID (if not already present).
  2. Create a Services ID (e.g., dev.constans.threadscout).
  3. Enable Sign In with Apple.
  4. Configure the Primary App ID and register the Redirect URI.
  5. Generate a Private Key (.p8) and record the Key ID and Team ID.

eBay (eBay Developer Program)

  1. Log in to the eBay Developer Portal.
  2. Navigate to Application Keys.
  3. Click User Consent for the Production environment.
  4. Provide the Redirect URI. eBay will generate a RuName (Redirect URL Name).
  5. Note: The RuName must be stored in fnox as THREADSCOUT_EBAY_RU_NAME.

GitHub (GitHub Developer Settings)

  1. Navigate to Settings > Developer Settings > OAuth Apps.
  2. Create a New OAuth App.
  3. Homepage URL: https://threadscout.constans.dev
  4. Authorization callback URL: See Table above.

4. Post-Registration: Secret Storage

Once you have obtained the Client ID and Client Secret, they must be securely stored in fnox.

Command Template:

fnox set --profile production THREADSCOUT_GOOGLE_CLIENT_ID "your-id"
fnox set --profile production THREADSCOUT_GOOGLE_CLIENT_SECRET "your-secret"

Refer to guides/deployment.md for the full list of secret keys.

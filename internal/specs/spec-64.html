SPEC-64: Integrated Ingress Automation

1. Goal

Automate the setup and management of “Local-to-Public” routing by integrating tunnel services directly into the development workflow.

2. Traceability

  • Phase: Phase 8: Production Readiness (DX/Infra).
  • Objective: Resolve the “Reachability” problem for OAuth callbacks and Mobile Ingest (UJ-06).

3. Implementation Strategy

  1. Tunnel Integration: Add support for Tailscale Funnel or Cloudflare Tunnel directly into the mise dev task.
  2. Dynamic Callback Update: If a public tunnel is active, automatically update the REDIRECT_URI environment variable for OAuth providers.
  3. Zero-Config Reachability: Ensure that starting the stack automatically results in a reachable, HTTPS-enabled URL without manual DNS configuration.

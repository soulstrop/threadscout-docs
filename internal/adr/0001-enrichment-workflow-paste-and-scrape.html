ADR 001: Item Enrichment Workflow via Paste & Scrape

Status

Accepted

Context

The goal is to allow users to enrich extracted item metadata (Brand, Model, Material, etc.) by using external research. After performing a Google keyword search for images, the user needs to find a matching product page and pull that detailed information back into ThreadScout to improve the accuracy of the eBay Sold comparables search and final listing creation.

We evaluated three primary options:

  1. Paste & Scrape: User pastes a URL from their research into the app; the backend scrapes the page and uses AI to extract relevant details.
  2. Integrated Search: Bringing search results directly into the app via API.
  3. Browser Extension: A plugin to send data from resale sites directly to ThreadScout.

Decision

We chose Option 1: Paste & Scrape.

Users will find a matching listing in a standard browser tab, copy the URL, and paste it into a “Research URL” field in ThreadScout. The backend will then fetch the HTML content, and an LLM (Gemini) will be used to extract highly specific garment details (e.g., technical names, style codes, MSRP) from the raw page text.

Rationale

  • Robustness: Most retail and resale sites have aggressive anti-bot measures. By having the user navigate to the page first, we ensure the link is valid and accessible.
  • Accuracy: Human verification of a “best match” is significantly more reliable than automated image matching for specialized clothing items.
  • Low Complexity: This avoids the high development cost of a browser extension and the recurring API costs/scraping challenges of integrated Google Search.
  • AI Flexibility: Using Gemini to parse the scraped HTML allows the system to support a wide variety of sites (eBay, Grailed, Poshmark, Retailers) without writing site-specific scrapers.

Consequences

  • Positive: High accuracy of enriched data; support for almost any source URL; simplified backend logic.
  • Negative: Adds a manual “copy/paste” step to the user workflow.
  • Future Work: Requires implementation of a robust scraping endpoint and refined AI prompts for structured extraction from unstructured HTML.

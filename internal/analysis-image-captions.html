Feature Analysis: AI-Generated Image Captions

1. Problem Statement

Currently, images pushed to eBay are sent without captions or alt-text. Professional resellers often manually add keywords like “Front View”, “Close-up of Tag”, or “Known Flaw” to improve search SEO and buyer clarity. Doing this manually for 10+ photos per item is a significant “cognitive tax.”

2. Proposed Solution

Leverage the existing AI Vision step (Gemini 2.0 Flash) to not only extract item metadata but also classify and caption each individual image in the selection.

User Experience

  1. Analysis: When the user clicks “Analyze”, the AI returns both the item details (Brand, Size, etc.) and a mapped list of captions for every selected image.
  2. Review: The Listing Preview or Synthesis Editor displays these captions under each thumbnail for user verification/editing.
  3. Push: When pushing to eBay, these captions are included in the imageUrls or product.image metadata sent to the eBay Inventory API.

3. Technical Requirements

Backend (Python/FastAPI)

  • Model Update: Add a caption (String) field to the Image SQLModel.
  • Prompt Update: Modify Settings.prompt_analysis to include an image_captions field in the expected JSON response.
    • Example Structure: {"image_captions": {"file_1.jpg": "Front view of shirt", "file_2.jpg": "Detail of fabric content tag"}}
  • eBay Client: Update _prepare_inventory_item_data in api/ebay.py to include the imageUrls with their respective captions.

Frontend (Next.js/TypeScript)

  • State Management: Update ListingContext to store the AI-generated captions in the Listing interface.
  • UI: Update the SynthesisEditor or a new GalleryManager component to allow users to manually override the AI-generated captions before finalization.

4. Expected Benefits

  • SEO: Improved visibility in eBay search results via richer image metadata.
  • Transparency: Clearer identification of flaws or specific details for buyers, potentially reducing return rates.
  • Velocity: Maintains the “3-Click” rule by automating a task that would otherwise take minutes of typing per item.

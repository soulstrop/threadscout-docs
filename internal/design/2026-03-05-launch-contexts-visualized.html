This document illustrates the different networking paths and security boundaries for the various ThreadScout launch profiles.

Key Concepts

TLS Termination

  • Local: Caddy generates its own certificates. Fine for desktop browsers where you can click “Advanced -> Proceed”, but problematic for mobile.
  • Tailnet-HTTP: Bypasses local TLS to avoid mobile handshake errors. Secure because Tailscale encrypts the transport layer via WireGuard.
  • Public: Cloudflare handles the “World-to-Edge” encryption. The “Edge-to-Workstation” path is secured by the Cloudflare Tunnel.

The “Circle of Trust”

The tailnet-http profile assumes that any device capable of reaching gmktec:8080 is already authenticated to your private Tailscale network, allowing for rapid iteration without managing local certificates.
